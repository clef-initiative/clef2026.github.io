Touché is a series of scientific events and shared tasks on argumentation systems.

<!--more-->

## Tasks

The Touché Lab features four different tasks:

{{< accordion  open_all="true" >}}
  {{< accordion-item title="Fallacy Detection" >}}
  Participants determine whether an argument follows a valid argument pattern or a fallacy.
  {{< /accordion-item >}}
  {{< accordion-item title="Causality Extraction" >}}
  Participants extract pro- and concausal claims from natural language text.
  {{< /accordion-item >}}
  {{< accordion-item title="Generalizability of Argument Identification in Context" >}}
  Participants predict whether or not sentences would be annotated as an argument under different annotation guidelines.
  {{< /accordion-item >}}
  {{< accordion-item title="Advertisement in Retrieval-Augmented Generation" >}}
  Participants detect and block advertisements in generated text.
  {{< /accordion-item >}}
{{< /accordion >}}
